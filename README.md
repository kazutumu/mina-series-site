# ミナの絵本シリーズ サイト

GitHub Pagesでそのまま公開できる静的サイトです。ビルド手順はありません。

## 現在の掲載範囲

- 本編: 1〜56冊目
- 研究ノート: 1〜5冊目
- 森研究所マガジン: 1〜3冊目
- 外伝: 1〜5冊目

掲載範囲は `books/MINA_KDP_PUBLICATION_STATUS.md` の販売中・公開済み記録に合わせます。制作済みでも、KDP公開確認がない作品は追加しません。

## ファイル

- `index.html`: 一覧ページ本体
- `styles.css`: 画面デザイン
- `script.js`: 本編・関連本の一覧データと表示処理
- `*.jpg`: 表紙画像。GitHub Pagesの公開ルートに合わせて、画像は `index.html` と同じ階層に置きます。
- `sitemap.xml`: 検索エンジン向けのサイトマップ
- `robots.txt`: クロール許可とサイトマップ位置の指定

## 公開メモ

Amazonへのリンクは、現時点では作品名での検索リンクにしています。各本の正式な商品ページURLが決まったら、リンク生成部分を商品ページURLに差し替えられます。検索結果がずれる本が出た場合は、`script.js` の `searchQuery` で個別に検索語を調整できます。
